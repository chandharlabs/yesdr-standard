# YesDR Releases

This page documents official releases of the **YesDR Standard**, including architectural updates, protocol enhancements, tooling, and reference implementations. Each release is versioned, dated, and aligned with the corresponding YesDR Technical Specifications (TS).

---

## Latest Release

### YesDR v1.0.0 — Initial Public Standard  
**Release Date:** January 2026  
**Status:** Stable  

**Highlights**
- First official release of the YesDR standard framework  
- End‑to‑end modular cellular architecture from **PHY to Core**  
- Native support for **AI‑driven radio intelligence**  
- Reference control protocols: **YACP** and **YSMP**  
- Core network function set including **YAMF, YSMF, YUPF, YUDM, YPCF, YCRF**, and others  
- Wireshark dissectors and sample PCAP files for validation and education  

**Included Components**
- YesDR architecture specification  
- Control‑plane and spectrum‑management protocols  
- Core function definitions  
- Reference tooling and documentation

## Versioning Policy

YesDR follows a structured semantic versioning scheme:

- **MAJOR** version (X.0.0): Architectural or standard‑level changes  
- **MINOR** version (0.Y.0): New features, protocols, or functions added  
- **PATCH** version (0.0.Z): Clarifications, fixes, and documentation updates  

Example: `v1.2.3`
