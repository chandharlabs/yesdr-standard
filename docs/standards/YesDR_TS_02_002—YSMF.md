**Scope**:
Defines session establishment, modification, and release procedures.

**Summary**:
YSMF manages data session lifecycle, IP address allocation, QoS flow control, and association between control-plane and user-plane entities. It plays a role analogous to the 5G SMF, while allowing simplified and extensible session models.
