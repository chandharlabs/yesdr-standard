## YESDR Technical Specifications Summary (Release v1.0.0)

| Specification | Scope | Summary | Link |
|---------------|-------|---------|------|
| **TS 00.001 — Overall Framework** | SDR-native cellular framework | Describes the SDR-centric design philosophy enabling full programmability of waveforms, protocols, and AI-assisted end-to-end cellular experimentation. | [3GPP TS 23.501 – YesDR-00-001-Overall](../assets/YesDR_01_001.pdf) |
| **TS 00.004 — Architecture** | End-to-end system architecture | Defines the modular YESDR architecture spanning PHY, RAN, and Core Network using SDR-based, fully software-defined functional entities. | [3GPP TS 23.501 – YesDR-00-004-Architecture](../assets/YesDR-00-004-architecture.pdf) |
| **TS 04.001 — YACP** | Access and registration signaling | Specifies procedures for device registration, authentication initiation, capability exchange, and mobility signaling between YUE, YBS, and YCore. | [3GPP TS 23.501 – YesDR_TS_04_001_YACP](../assets/YesDR_TS_04_001_YACP.pdf) |
| **TS 04.002 — YSMP** | Spectrum coordination signaling | Defines spectrum usage reporting, allocation commands, interference alerts, and coordination among spectrum management and RAN/core entities. | [3GPP TS 23.501 – YesDR_TS_04_002_YSMP](../assets/YesDR_TS_04_002_YSMP.pdf) |
| **TS 02.002 — YSMF** | Session management | Defines session establishment, modification, release, IP allocation, QoS flow control, and CP–UP association. | [3GPP TS 23.501 – YesDR_TS_02_002_YSMF](../assets/YesDR_TS_02_002_YSMF.pdf) |
| **TS 02.003 — YUPF** | User-plane data forwarding | Specifies packet routing, encapsulation, tunneling, QoS enforcement, and MEC offloading for experimental data paths. | [3GPP TS 23.501 – YesDR_TS_02_003_YUPF](../assets/YesDR_TS_02_003_YUPF.pdf) |
| **TS 02.004 — YUDM** | Unified data management | Defines centralized storage of subscriber, device, service profiles, authentication parameters, and capability information. | [3GPP TS 23.501 – YesDR_TS_02_004_YUDM](../assets/YesDR_TS_02_004_YUDM.pdf) |
| **TS 02.005 — YAUSF** | Authentication and security | Specifies authentication procedures, key derivation, and security context establishment for users and devices. | [3GPP TS 23.501 – YesDR_TS_02_005_YAUSF](../assets/YesDR_TS_02_005_YAUSF.pdf) |
| **TS 02.006 — YPCF** | Policy control | Defines policy generation and enforcement for QoS, access restrictions, traffic prioritization, and charging-related decisions. | [3GPP TS 23.501 – YesDR_TS_02_006_YPCF](../assets/YesDR_TS_02_006_YPCF.pdf) |
| **TS 02.009 — YNRF** | Network function registry | Specifies service registration and discovery of YESDR network functions, enabling dynamic discovery and service-based core interactions. | [3GPP TS 23.501 – YesDR_TS_02_009_YNRF](../assets/YesDR_TS_02_009_YNRF.pdf) |
| **TS 02.010 — YSM** | Spectrum monitoring and measurement | Defines continuous spectrum observation, interference detection, occupancy estimation, and reporting to spectrum control entities. | [3GPP TS 23.501 – YesDR_TS_02_010_YSM](../assets/YesDR_TS_02_010_YSM.pdf) |
| **TS 02.020 — YBS** | Base station (RAN node) behavior | Specifies radio access, scheduling, timing, and control signaling between YUE and the core network, enabling experimental RAN procedures. | [3GPP TS 23.501 – YesDR_TS_02_020_YBS](../assets/YesDR_TS_02_020_YBS.pdf) |
| **TS 02.021 — YUE** | User equipment behavior | Defines PHY, MAC, RLC, PDCP, and control-plane behavior for SDR-based user devices, supporting experimentation and AI-assisted PHY processing. | [3GPP TS 23.501 – YesDR_TS_02_021_YUE](../assets/YesDR_TS_02_021_YUE.pdf) |
